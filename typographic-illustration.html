Typographic Illustration

About the Project

Typographic illutstration of the Mad Hatter in black and white. This character belongs to the Disney's Alice in Wonderland movies. The goal with this project was to showcase typography only. From the hat, to the blazer, the only rule was to fill the figure using font elements. Artwork created in Adobe Illustrator using a reference picture of the real character portrayed by Jhonny Deep.
