Scientific Illustration

About the Project

Scientific illustration of the South American bird Hyacinth Macaw. The goal with this project was to showcase a realistic version of an animal. From the fur, to the tree trunk, the only rule was to create the figure as realistic as possible. Artwork created in Adobe Illustrator using a reference picture of the real blue macaw.
